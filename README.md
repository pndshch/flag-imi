# 世界の国旗の意味 / Meaning of World Flags

世界197の国と地域の国旗を収録し、色とシンボルに込められた意味を日本語・英語で解説するブラウザリファレンス。

A bilingual (Japanese / English) browser reference explaining the colours and symbols of 197 national flags.

**Live:** https://pndshch.github.io/flag-imi/

## 構成 / Structure

- `index.html` — アプリ全体（バニラ HTML/CSS/JS、ビルド不要 / vanilla HTML/CSS/JS, no build step）
- `flags/` — 国旗 SVG（パブリックドメイン。出典は `flags/LICENSE-flags.txt` / public-domain flag SVGs, provenance in `flags/LICENSE-flags.txt`）

各国旗について、地域・縦横比・制定年と、色ごと・シンボルごとの意味を「公式」と「通説」に分けて表示します。

Each flag entry shows its region, aspect ratio and adoption year, plus the meaning of every colour and symbol — distinguishing officially documented meanings from traditional ones.

Part of [Pinda Lab](https://pndshch.github.io/).
